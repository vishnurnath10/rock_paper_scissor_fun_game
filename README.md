# Rock-Paper-Scissors Game

A simple **Rock-Paper-Scissors** game in Python with ASCII art.  
This project is a beginner-friendly mini game to practice Python basics, including:

- Taking user input
- Using `if-elif-else` statements
- Using the `random` module
- Lists and mapping for ASCII art

## How to Play

1. Run the `main.py` file.
2. Enter your choice:
   . `0` for Rock  
   . `1` for Paper  
   . `2` for Scissors
3. The computer will randomly choose.
4. The winner will be displayed.
